# Choices: A Narrative Journey

An interactive narrative game exploring how personal choices are shaped by gender, culture, family expectations, and social norms.

## How to Play

1. Select your character background (Gender, Cultural Background, Family Expectation Level)
2. Make choices as the story unfolds
3. Experience how your background shapes the narrative and available options
4. Reflect on the ending summary

## Setup for GitHub Pages

To make this playable online:

1. Go to your repository on GitHub: https://github.com/ShaheerSaud2004/Gender-Project
2. Click on **Settings** (in the repository menu)
3. Scroll down to **Pages** (in the left sidebar)
4. Under **Source**, select **Deploy from a branch**
5. Choose **main** branch and **/ (root)** folder
6. Click **Save**
7. Wait a few minutes for GitHub to deploy
8. Your game will be available at: `https://shaheersaud2004.github.io/Gender-Project/`

The game is a single HTML file with embedded CSS and JavaScript - no backend needed!
